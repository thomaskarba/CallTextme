Steps To Set Up Karba Layout Keyboard on PC

For Mac Please click here

1. Download and install the following software: SharpKeys39.msi

2. Download the keys for Sharpkeys: Karba_layout_SharpKeys.skl

3. Open SharpKeys, load keys, and click ' Write to Registry'.

4. Restart your PC.
